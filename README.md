Este portal de notícias foi criado com o objetivo de servir como um espaço dedicado à publicação de conteúdos relacionados aos meus estudos, especialmente nas áreas de tecnologia e desenvolvimento.
# Arquitetura front-end com next.js e WordPress headless
O front-end foi desenvolvido utilizando Next.js e React, juntamente com as demais tecnologias listadas nas tags do projeto. Um dos principais pontos técnicos foi a integração entre o WordPress e o front-end desacoplado, adotando uma abordagem headless CMS.

O WordPress foi utilizado como backend, aproveitando sua estrutura nativa de gerenciamento de conteúdo (CMS), o que permitiu administrar publicações de forma eficiente sem a necessidade de desenvolver um sistema próprio do zero. 

O front-end consome a API REST padrão do WordPress, utilizando os endpoints de posts para recuperar e renderizar dinamicamente dados como imagens destacadas, categorias, autores e demais informações relacionadas ao conteúdo. 

# Instrução de instalação
Siga os passos abaixo para rodar o projeto localmente:
#### Clone do repositório
```bash
git clone https://github.com/joaovictor25/portal-de-noticias.git

```
#### Acessar diretorio do projeto
```bash
cd base-tecnologia
```
#### Instalar as dependências
```bash
npm install
```
#### Executar projeto
```bash
npm run dev
```
#### Acessar o projeto
```bash
http://localhost:3000
```
