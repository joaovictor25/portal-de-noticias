Este portal de notícias foi criado com o objetivo de servir como um espaço dedicado à publicação de conteúdos relacionados aos meus estudos, especialmente nas áreas de tecnologia e desenvolvimento.

# Instrução de instalação
Siga os passos abaixo para rodar o projeto localmente:
#### Clone do repositório
```bash
git clone https://github.com/joaovictor25/portal-de-noticias.git

```
#### Acessar diretorio do projeto
```bash
cd base-tecnologia
```
#### Instalar as dependências
```bash
npm install
```
#### Executar projeto
```bash
npm run dev
```
#### Acessar o projeto
```bash
http://localhost:3000
```
